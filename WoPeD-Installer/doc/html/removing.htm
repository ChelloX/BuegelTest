Removing Petri Net Elements

To remove a net element, right-click on the element and select "Edit -> Remove" from the context menu.

Note that removing a place or transition element will also automatically remove all adjacent arcs.

Removing triggers

In order just to remove a Trigger from a transition, right-click on the trigger symbol and select "Edit -> Trigger ->   None" from the context menu.

Removing tokens

In order just to remove a Token from a place, right-click on the place symbol and select "Edit -> Token ->   Remove" from the context menu. Tokens can only be removed one-by-one, so in order to remove more than one token, the described operation must be applied repeatedly.
