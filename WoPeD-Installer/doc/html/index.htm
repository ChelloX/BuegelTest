WoPeD Manual

WoPeD (Workflow Petri net Designer) is an easy-to-use, platform-independent software tool for modelling and analyzing business process descriptions in terms of a special class of Petri nets called Workflow Nets.

Downloading WoPeD

The newest release of WoPeD can be downloaded on the WWW either from the WoPeD homepage or from the Sourceforge project site. Before installing WoPeD, please download and install the Java Runtime Environment (JRE - Version 1.4.2 or higher) on your computer. Java can be obtained from the Sun Java homepage.

Installing WoPeD

Now you are ready to install WoPeD. Please unpack the downloaded archive and run the contained executable program WoPeD-install.exe (in case of a Windows-based system) or WoPeD-install (in case of a Unix-based system). In both cases, a wizard will guide you through the setup procedure - just follow the instructions.

Starting WoPeD

After successful installation, you will find shortcuts on the desktop and in the start menu (in case of a Windows-based system and for most - maybe not all - UNIX GUI systems) to start WoPeD. Alternatively, you can start WoPeD either by starting the executable file WoPeD.exe (in case of Windows) or WoPeD (in case of UNIX) in the installation target folder. You may also want to associate the file extension "*.pnml" to WoPeD, letting you start the program with directly opening a given Petri net file by just double-clicking on it.

Using WoPeD

After starting WoPeD, the main window becomes visible, containing the main components of the WoPeD user interface:

  • Menubar
  • Toolbar
  • Drawing Area
  • Statusbar
