Files Configuration

In the files group, the following system settings can be defined:

Default directory

Select the folder to which all file operations (open, save, save as, export, ...) automatically shall refer to. If the box "Use default directory" is checked, this folder is supposed to be "nets" within the installation path of WoPeD. To change this path, just uncheck the box and type in a full path name or use the "Browse" button to search for it.

Note that you must have write access to this directory!

Delete recent files

Press this button to erase the contents of the "Recent" list from the "File" menu.

PNML import/export

Check these boxes for using the extended Workflow Net notation instead of the standard Petri Net notation in PNML files. Otherwise the extended constructs will be ignored. It is usually not recommended to disable these options.

TPN export

When using Woflan and exporting the current net into the associated TPN format, the names of net elements will be included in the element descriptions of TPN, making it easier to understand Woflan's diagnostic messages. It is recommended to leave this option checked.
