Places

Places are the passive elements of a Petri net. Places can contain zero, one or more Tokens representing data which is stored on the place.

Creating

To create a new place in an open Editor use one of the following commands:

  • Left-click on the associated toolbar symbol   and then to the position where the place shall appear
    or
  • Right-click at the position where the place shall appear and choose the command "Add -> Place" from the context menu

After this, you can fill-out the small editing box to assign a name to the new place or leave this to the automatic naming mechanism of WoPeD.
