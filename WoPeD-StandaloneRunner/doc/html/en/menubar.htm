Menubar

Unless explicitly mentioned, all menu commands refer to the active editor window. The main window menubar has the following entries:

File menu

  •   New - open a new, empty editor
  •   Open - load an existing Petri net from file
  •   Recent submenu - list of recently opened files
  •   Close - close the active editor
  •   Save - save the currently edited Petri net to a file
  •   Save as - save the currently edited Petri net to a file with a new name
  •   Print - print currently edited Petri net
  •   Export - export the current net (currently supported: PNG, JPG, BMP and BPEL format)
  •   AProMoRe Import
  •   AProMoRe Export

Edit menu

  •   Undo - undo the last action
  •   Redo - undo the last undone action
  •   Cut - cut the selected items to the clipboard
  •   Copy - copy the selected items to the clipboard
  •   Paste - paste the clipboard contents into the currently edited Petri net
  •   Group - group the selected items to a compound item
  •   Ungroup - split a previously grouped, compound item into its components
  •   Draw a place
  •   Draw a transition
  •   Draw an AND-split transition
  •   Draw an XOR-split transition
  •   Draw an AND-join transition
  •   Draw an XOR-join transition
  •   Draw an XOR-split-join transition
  •   Draw an AND-split-join transition
  •   Draw an AND-join XOR-split transition
  •   Draw a subprocess transition

Analyze menu

  •   Start tokengame - starts the tokengame
  •   Operator coloring - effects a Coloring for the associated operators
  •   Semantical analysis - starts the Wizard
  •   capacity planning - opens capacity-planningdialog
  •   Quantitative analysis - starts the quantitative analysis
  •   Coverability Graph - starts the coverability graph
  •   Show metrics - opens the Process-Metrics Sidebar
  •   Metrics mass analysis - opens the Process-Metrics mass analysis
  •   Metrics builder - opens the Process-Metrics builder

View menu

  •   Flip modeling direction - switches the modeling direction between vertical and horizontal
  •   Optimize layout (multiple executions lead to different results)
  •   Cascade windows - show all open editor windows overlapping
  •   Arrange windows - show all open editor windows without overlapping
  •   Overview - show the overview in the sidebar
  •   Treeview - show the treeview in the sidebar

Options & Help menu

  •   Configuration - open configuration dialog for global settings
  •   Manual - open help browser start page
  •   Contents - open help browser contents overview
  •   Sample nets - list of sample files (to be opened read-only)
  •   Report Bug - open bug report page
  •   About - open about dialog
