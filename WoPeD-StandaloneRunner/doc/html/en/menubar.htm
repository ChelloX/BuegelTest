Menubar

Unless explicitly mentioned, all menu commands refer to the active editor window. The main window menubar has the following entries:

File menu

  •   New - open a new, empty editor
  •   Open - load an existing Petri net from file
  •   Close - close the active editor
  •   Save - save the currently edited Petri net to a file
  •   Save as - save the currently edited Petri net to a file with a new name
  •   Print - print currently edited Petri net
  •   Export - export the current net (currently supported: TPN, PNG, JPG, BMP and BPEL format)
  •   Recent submenu - list of recently opened files
  •   Quit - terminate WoPeD

Edit menu

  •   Undo - undo the last action
  •   Redo - undo the last undone action
  •   Cut - cut the selected items to the clipboard
  •   Copy - copy the selected items to the clipboard
  •   Paste - paste the clipboard contents into the currently edited Petri net

View menu

  •   Group - group the selected items to a compound item
  •   Ungroup - split a previously grouped, compound item into its components
  •   Zoom in - enlarge the Petri net graphics by 10 %
  •   Zoom out - shrink the Petri net graphics by 10 %
  •   Rotate Transition Left - rotates the selected transition counterclockwise
  •   Rotate Transition right - rotates the selected transition clockwise
  •   Change modeling direction - switches the modeling direction between vertical and horizontal
  •   Default: automatic layout-optimizing (multiple executions lead to different results)
  •   Advanced: forgiveness-counter and grid-spacing is editable (multiple executions lead to different results)

Analyze menu

  •   Start tokengame - starts the tokengame
  •   Call Woflan - starts Woflan analysis
  •   Semantical analysis - starts the Wizard
  •   capacity planning - opens capacity-planningdialog
  •   Quantitative analysis - starts the quantitative analysis
  •   Reachability Graph - starts the reachability graph
  •   Process Metrics - opens the Process-Metrics Sidebar
  •   Mass Process Metrics Analysis - starts the Mass Process Metrics Analyser
  •   Metrics Builder - starts the Metrics Builder

Options menu

  •   Configuration - open configuration dialog for global settings

Window menu

  •   Cascade - show all open editor windows overlapping
  •   Arrange - show all open editor windows without overlapping
  •   Show editor sidebar - shows the sidebar on the currently selected editor
  •   List of open windows (if present)

Help menu

  •   Manual - open help browser start page
  •   Contents - open help browser contents overview
  •   List of sample files (to be opened read-only)
  •   About - open about dialog
  •   Report Bug - open bug report page
