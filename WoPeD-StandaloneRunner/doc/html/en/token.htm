Tokens

Tokens represent data elements assigned to Places. Each place can have zero, one or more tokens on it.

In order to add a token to a place, right-click on the place symbol and select "Edit -> Token ->   Add" from the context menu. Tokens can only be added one-by-one, so in order to add more than one, the described operation must be applied repeatedly.

In order to remove a token from a place, right-click on the place symbol and select "Edit -> Token ->   Remove" from the context menu. Tokens can only be removed one-by-one, so in order to remove more than one token, the described operation must be applied repeatedly.
