Coverability Graph

The reachability of a petri net's states can be represented with a coverability graph, a directed graph whose points represent states (i.e. M) and arcs represent transitions between two such states. The graph is constructed through breadth-first search as follows: the starting state (M0) is taken, and all possible transitions are explored from this state, then the transitions from these states, and so on. As the coverability graph may be infinitely large, breadth-first search is preferred because depth-first search would not find all possible states even if given infinite time.
(This definition was taken from wikipedia.org)

 

Start Coverability Graph

Start the coverability graph calculation by clicking the - button in the tool- or menubar. The button is only activated if an editor window is opened and focused.

WoPeD creates only one additional window for all the coverability graphs. All editors share this single coverability graph window. In order to do that, the coverability graph window will automatically be filled with the coverability graph of the focused editor window.

The initial marking of a coverability graph is always in a grey-scaled but you can choose a colored mode in the settings.

Coverability Graph Window

1 Title

The name of the current shown editor is displayed here.

2 Graph Drawing Place

The main area of the window consists of a graph drawing area where the graph is displayed. If the resulting graph is bigger than the drawing area, it is surrouded by scroll bars. (Behaviour is analog to a normal editor window.)

3 Toolbar

3.1 Refresh - Button
The refresh button can only be activated when the underlying petri net has changed. In order to not to accidentally destroy a manual layout the refresh has always to be done manually by the user.
3.2 Unset - Button

With this button you can unset the state-highlighting. This can also be done by double-click in the background of the coverability graph.

3.3 Settings - Button
By clicking the settings button a dialog opens. There you can edit some preferences of the coverability graph.
3.4 Export as - Button

By clicking the export as button a filechooser dialog opens. There you can export the current coverability graph as a picture in different file-formats.

3.5 Layoutchooser
With the layout chooser you can select a layout.
3.6 Zoom

You can adjust the zoom-level by clicking the buttons to get 10 percent more or less. Some standard zoom-levels can be selected in the box or a value from 20 to 100 can be typed in.

4 Statusbar

The statusbar shows information that belongs to the current coverability graph.
