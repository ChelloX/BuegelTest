Reachability Graph

The reachability of a petri net's states can be represented with a reachability graph, a directed graph whose points represent states (i.e. M) and arcs represent transitions between two such states. The graph is constructed through breadth-first search as follows: the starting state (M0) is taken, and all possible transitions are explored from this state, then the transitions from these states, and so on. As the reachability graph may be infinitely large, breadth-first search is preferred because depth-first search would not find all possible states even if given infinite time.
(This definition was taken from wikipedia.org)

 

Start Reachability Graph

Start the reachability graph calculation by clicking the [W] - Button in the toolbar. The Button is only activated if a editor window is open and focused.

WoPeD will create only one additional window for the reachability graphs. All editors share this single reachability graph window. In order to that, the reachability graph window will automatically be filled with the reachability graph of the focused editor.

The initial marking of a reachability graph is always in green color (except in gray-scale mode). All other nodes are orange.

Reachability Graph Window

1 Title

In title is displayed the name of the actual shown editor.

2 Graph Drawing Place

The main area of the window consists of graph drawing area. There the graph is displayed. If the resulting graph ist bigger than the drawing place, it's surrouded by scroll bars. (Behaviour is analog to a normal petri net editor window.)

3 Toolbar

3.1 Refresh - Button
The refresh button is only active when the underlying petri net has changed. To not accidentally destroy a manual layout the refresh is always done manually by the user.
3.2 Layout Chooser
With layout chooser you can choose a layout.
3.3 Settings - Button
By clicking the settings button a dialog opens. There you can edit some preferences to the reachability graph.
3.4 Export as - Button
By clicking the export button a filechooser dialog opens. There you can export the actual reachability graph as picture in different file-types.

4 Statusbar

The statusbar shows information that belong to the actual reachability graph.
