Changing Element Attributes

Double-clicking on a net element opens a dialog box allowing to change all available attributes interactively. Be sure to perform the double-click on the center area of the element where the mouse cursor is shown as a pointing finger. Besides this, some attributes of Petri net elements can be changed using alternative commands.

Changing a name

In order to change the name of a net element, right-click on the name tag and select "Edit ->   Edit name" from the context menu. A small editing panel appears and lets you type in the new name. To finish editing, use the <return> key.

Changing a trigger

In order to change the Trigger of a transition, right-click on the transition symbol, select "Edit -> Trigger" from the context menu and choose the desired trigger type ("Message", "Resource", "Time", "None").

Adding a token

In order to add a Token to a place, right-click on the place symbol and select "Edit -> Token -> Add" from the context menu. Tokens can only be added one-by-one, so in order to add more than one, the described operation must be applied repeatedly.
