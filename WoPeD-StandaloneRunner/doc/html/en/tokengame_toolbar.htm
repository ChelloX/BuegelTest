Tokengame Toolbar

General remarks

There are two methods for tokengame simulation: Use the toolbar and interact within the net or take the tokengame remotecontroller.

Tokengame Remotecontroller

To start the tokengame remotecontroller just select one in the drop-down menu of the toolbar.

Tokengame Toolbar

If you have designed a petrinet and set a number of tokens you can activate tokengame simulation by using Tokengame symbol on main menubar. When Tokengame simulation is started you can not change the petrinet anymore till simulation is stopped again. Themain toolbar disappears and simumlation toolbar appears at the same place. The TokenGame is started automatically.

Playback Controls

  • FastBackward - Jump to the next decision point
  • Backward - apply the next transition
  • StepIntoSubprocess
  • Play - starts the TokenGame
  • StepOverSubprocess
  • Pause
  • Stop
  • FastBackward - Jump to the next decision point
  • Backward - apply the next transition

Playback-ChooseTransition-Dialog

If there is a transition to choose and the mousepointer is over the playback-Control there opens a modal dialogbox to choose the transition. The box is open as long as there is something to decide.

Properties-Controls

  • stepwise playback - default
  • autoplayback
  • Playback-Properties Dialog
  • autoTransitionOccurs – Chooses the next transition automatically

Stop the Tokengame

The   button stops the SimulationMode and switches to the standard menu toolbar
