Editor configuration

In the editor group, the behaviour of the editor can be customized.

Smart edit

This feature makes the creation of nets more productive by minimizing the number of mouse clicks needed. There are two options:

  • Check "Smart name editing" to enable jumping automatically to editing the name of a newly-created net element
  • Check "Smart arc drawing" to enable creation of net elements "on-the-fly" by just following outgoing arcs

Additionally, check "Add '_copy' to copied elements" to adjust the automatic naming rule.

Grid

The grid makes positioning net elements easier. Check the box "Show grid" to show the grid point the drawing area. To set the grid distances to another value, choose the associated number from the chooser.

Arcs

The arc routing mode can be customized here. Select "Rounded" to let arcs flow around their arc points smoothly or "Straight" to let them run unbended along their arc points. Additionally, the line width, the arrow head size and the arrow filling option can be adjusted.
