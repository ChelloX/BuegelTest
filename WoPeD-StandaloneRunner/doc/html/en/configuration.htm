WoPeD configuration

By selecting Options -> Configuration from the main menubar or choosing the  Button from the toolbar, a dialog box appears in which several system-wide parameters of WoPeD can be adjusted. They are arranged in four groups:

  • Editor configuration
  • Files configuration
  • Tools configuration
  • Language configuration


To choose a group and access the associated options, just click on the group name in the left column pane.

To apply the changes made within one group, press the " Apply" button. To apply all settings and close the dialog window, press " Ok". To dismiss all changes and close the dialog window, press " Cancel".
