Metrics configuration

Use metrics

Enables or disables metric loading at WoPeD startup.



Decimal places

Number of decimal places that will be displayed for variables or algorithms in the Metrics Sidebar.



Show metric names instead of IDs

Switch to show the more human readable metric names in the lists of the Metrics Builder, rather than the more technical IDs.



Show advanced error messages

Option to enable more detailed and advanced error message in the Metrics Builder. Not needed for most users.
