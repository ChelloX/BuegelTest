Resource class definiton

In the upper part of the resource editor tab, resource classes (roles or groups) can be added, modified and deleted.

Add a resource class

In order to define a new resource class, click on " New", enter its name in the associated input pane, choose the associated type (role or group) from the selection list and click on " Ok" to save it. The resource class will then appear in the left-hand list pane. Note that you have to choose unique names for all resource classes of your model.

Modify a resource class

In order to change an existing resource class, click on the associated item in the left-hand list pane. It will appear in the input pane and can be modified there (name and/or type). When finished, click on " Ok" to save the changes. Note that you cannot change the type of a resource class as long as is used in the process model.

Delete a resource class

In order to delete an existing resource class, click on the associated item in the left-hand list pane. When it appears in the input pane, click on " Delete" to remove its definition. Note that you cannot delete a resource class as long as it is used in the process model or has resource objects assigned.
