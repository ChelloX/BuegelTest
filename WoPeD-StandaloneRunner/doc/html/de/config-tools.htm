Tools Configuration

The only option available in the tools group is setting the path to Woflan in order to invoke it directly from WoPeD, e. g. in order to check an edited net for soundness. If the box "Use Woflan" is checked, the associated path of the Woflan executable wofapp.exe can be typed in or searched via the "Browse" button.

  Note that Woflan is only available for Microsoft Windows.
