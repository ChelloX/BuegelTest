Resource object definiton

In the lower part of the resource editor tab, resource object (individual resources) can be added, modified, deleted and assigned to resource classes.

Add a resource object

In order to define a new resource object, click on " New", enter its name in the associated input pane and click on " Ok" to save it. The resource object will then appear in the left-hand list pane. Note that you have to choose unique names for all resource objects of your model.

Modify a resource object

In order to change an existing resource object  click on the associated item in the left-hand list pane. It will appear in the input pane and can be modified there. When finished, click on " Ok" to save the changes.

Delete a resource object

In order to delete an existing resource object, click on the associated item in the left-hand list pane. When it appears in the input pane, click on " Delete" to remove its definition.

Assign a resource object to resource classes

In order to define the set of resource classes an object belongs to, click on the item in the left-hand list pane. Use the " Assign" and " Remove" buttons to specify the set of resource classes the associated object shall belong to.
