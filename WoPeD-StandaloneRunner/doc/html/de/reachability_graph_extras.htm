Reachability Graph Extras

Coverability

Computing a reachability graph can be very difficult to handle. Some graphs tend to get very big or are not really solvable in available time. There is a special case of reachability graphs that are not ending while computing them. Due to the unboundedness of a petri net. An unbounded petri net has places, that can get an infinite number of tokens.
To handle this situation and to still get the graph drawn you can compute a coverability graph. This graph is just like a normal reachability graph but it can handle unbounded places. It represents unbounded places by a "w".
