Toolbar

Unless explicitly mentioned, all toolbar commands refer to the active editor window. The main window toolbar contains the following entries:

  •   Open a new, empty editor
  •   Load an existing Petri net from file
  •   Save the the currently edited Petri net to a file
  •   Save the the currently edited Petri net to a file with a new name
  •   Cut the selected items to the clipboard
  •   Copy the selected items to the clipboard
  •   Paste the clipboard contents into the currently edited Petri net
  •   Undo the last action
  •   Redo the last undone action
  •   Group the selected items to a compound item
  •   Split a previously grouped compound item into its components
  •   Zoom up the Petri net graphics by 10 %
  •   Zoom down the Petri net graphics by 10 %
  •   Draw a place
  •   Draw a transition
  •   Draw an AND-split transition
  •   Draw an XOR-split transition
  •   Draw an AND-join transition
  •   Draw an XOR-join transition
  •   Draw an XOR-split-join transition
  •   Draw an AND-split-join transition
  •   Draw an AND-join XOR-split transition
  •   Draw a subprocess transition
  •   Start or close a Token Game simulation
  •   Start Woflan analysis
  •   Start semantical analysis
  •   Reachability Graph - starts the reachability graph


In order to get a short description of the function of any toolbar button, just stay with the mouse pointer on it for a few seconds until an explanative text will appear in a small window.
