Resource class definiton

In the right part of the resource editor tab, resource classes (roles or groups) can be added, modified and deleted.

Add a resource class

In order to define a new resource class, click on "" button in the role or in the group panel, enter its name in the associated input window and click on "Ok" to save it. The resource class will then appear in the list pane. Note that you have to choose unique names for all resource classes of your model.

Modify a resource class

In order to change an existing resource class, click on the associated item in the list pane and then then click on the edit button "" . A window will appear and the name can be modified there. When finished, click on "Ok" to save the changes.

Delete a resource class

In order to delete an existing resource class, click on the associated item in the list pane. When you have marked the item, click on "" to remove it. Note that you cannot delete a resource class as long as it is used in the process model or has resource objects assigned.
