Elemente entfernen

Um ein Element aus einem Netz zu entfernen, kann man mit der rechten Maustaste auf das Element klicken und im Kontextmenü den Eintrag " Löschen ".

  Achtung, das entfernen eines Element führt automatisch zum entfernen der mit dem Element verbunden Pfeile.

Entfernen von Triggerelementen

Um ein Triggerelement vonm einer Transiton zu entfernen, kann man nach einem Rechtsklick auf das Triggerelement im Kontextmenü "  Kein Ttrigger" auswählen. Alternativ ist es möglich die Triggereinstellungen innerhalb des Transitionseinstellungen Dialoges zu ändern. Dieser wird erreicht wenn man nach dem Rechtsklick " Einstellungen" im Kontextmenü auswählt oder einen Doppelklick in der Mitte der Transition ausführt..

Entfernen von Marken

Das Entfernen einer Marke von einer Stelle kann ebenfalls nach einem Rechtsklick augeführt werden. Wählen sie hierzu dein Eintrag " Marke entfernen" aus. Marken können nur einzeln entfernt werden, um also mehrere Marken zu entfernen müssen sie den Vorgang mehrfach ausführen. Alternativ ist es möglich die Anzahl der Marken im Dialog Stelleneigenschaften zu ändern. Dieser wird erreicht wenn man nach dem Rechtsklick " Einstellungen " im Kontextmenü auswählt oder durch Doppelklick auf die Mitte der Stelle.
