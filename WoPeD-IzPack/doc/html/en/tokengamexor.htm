Choosing an XOR transition to cccur

When an XOR-split or XOR-join transition is enabled in the token game, it can be chosen for occurrence by just clicking on one of the associated arcs.

In case of an XOR-split, the input tokens will be removed and one output token will be added according to the place connected to the selected arc. In case of an XOR-join, one input token will be removed from the place connected to the selected arc and the output tokens will be added.

After this, the token game algorithm searches for the next set of enabled transitions.
