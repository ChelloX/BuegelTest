Removing elements

To remove a net element, right-click on the element and select " Delete" from the context menu.

  Note that removing a place or transition element will also automatically remove all adjacent arcs.

Removing triggers

In order just to remove a trigger from a transition, right-click on the trigger symbol and select "  No trigger" from the context menu. Alternatively, trigger properties can be edited inside the transition property dialog which can be opened by selecting " Properties" from the context menu or double-clicking on the element center area.

Removing tokens

In order just to remove a token from a place, right-click on the place symbol and select " Remove token" from the context menu. Tokens can only be removed one-by-one, so in order to remove more than one token, the described operation must be applied repeatedly. Alternatively, token properties can be edited inside the place property dialog which can be opened by selecting " Properties" from the context menu or double-clicking on the element center area.
