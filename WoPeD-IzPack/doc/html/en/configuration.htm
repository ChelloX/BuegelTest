WoPeD configuration

By selecting Options & Help in the menubar and the icon, a dialog box appears in which several system-wide parameters of WoPeD can be adjusted. They are arranged in six groups:

  • Editor configuration
  • Import / Export configuration
  • Language configuration
  • Coloring configuration
  • Metrics configuration
  • AProMoRe configuration


To choose a group and access the associated options, just click on the desired tab.

To apply the changes made within one group, press the " Apply" button. To apply all settings and close the dialog window, press " Ok". To dismiss all changes and close the dialog window, press " Cancel".
