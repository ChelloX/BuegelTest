BPEL Activity "invoke"

The invoke activity directs a Web service to perform a one-way or request-response operation. It specifies the partner link and operation to invoke. The invoke activity must specify an input variable for the message data to transmit and may specify an output variable or variable part in the case of a synchronous request-response Web service invocation.

 

To add an invoke activity to the process manually:

1. In the properties view, select the following values in order:

  1. Partner Link

  2. Operation: valid operations appear in the drop-down list after you select a partner link

2. Optionally, select an input variable from the drop-down menu or create one 

3. Optionally, select an output variable from the drop-down menu or create one
