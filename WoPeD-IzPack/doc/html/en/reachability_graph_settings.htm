Coverability Graph Settings Dialog

Overview

The settings dialog allows you to change the setting of the coverability graph. It's divided in 2 sections.

Graph Visuals

This section defines the visual appearance of the nodes in the coverability graph. It is possible to set the notation of markings, the color scheme and the height and the minimum width of one node. If the marking needs more space than the minimum width, the width gets adjusted automatically.

Hierarchic Settings

In the second section the horizontal and the vertical space between nodes can be adjusted. This settings only affect the hierarchical and the tree layout.
