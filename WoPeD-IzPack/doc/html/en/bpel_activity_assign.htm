BPEL Activity "assign"

The assign activity manipulates process variables by creating copy From/To operations for them.

 

The assign activity updates the content of variables. It updates variables by copying data from one variable to another.

 

To add an assign activity to the process:

1. In the From section create a variable of a specified type

2. In the To section create also a variable of a specified type

The To type must be compatible with the From type!!!
