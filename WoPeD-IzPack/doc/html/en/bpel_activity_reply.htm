BPEL Activity "receply"

The reply activity sends a response to a partner identified in a matching receive activity.

A reply activity sends a message back to a Web service in reply to a message received. It has the same partner link and operation as its matching receive.

To add a reply activity to the process manually:

1. In the properties view, select the following values in order:

  1. Partner Link

  2. Operation: valid operations appear in the drop-down list after you select a partner link

2. Optionally, select an variable from the drop-down menu or create one
