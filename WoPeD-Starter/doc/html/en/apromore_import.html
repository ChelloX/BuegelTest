AProMoRe Import

To import a process model from a Server, be sure that the Server-Informations at AProMoRe-Configurations are correct.

To import a process model, press the importbutton In the following there will be the AProMoRe-Import-Dialogue:

 

 

1 Search function

Here you can search for existing process models by entering a "Process name", "ID", "Owner", "Native Type", "Domain".

2 Current Server

The AProMoRe-Server which is in use is shown here.

3 List of process models

Here are shown the existing process models on the chosen server. The listing updates automatic when using the search function.
