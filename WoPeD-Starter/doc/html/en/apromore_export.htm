AProMoRe Export

Before exporting a model to an AProMoRe server, be sure that the AProMoRe configuration settings are correct.
For export, press the export button which will open the following dialog box:

 

 

1 Search function

Just type in sub-strings of the associated attributes "process name", "ID", "owner", "native type", and "domain". The entries int the list of process models will be narrowed accordingly.

2 Current Server

This shows the selected AProMoRe-Server. To change it, use AProMoRe configuration settings.

3 List of process models

Here the existing process models and their attributes on the chosen server are shown.

4 Process data

The fields are automatically filled whenever an entry in the list of process models is selected. The field can be edited to your needs, but please make sure to use an increased "Recent version" string.

To update the current editor contents as the selected process model (i. e. adding a new version of an existing model) click on "Update". To export the current editor contents to a new process model, just click on "Export".
