Text2Process

Description

You can open the Text2Process petrinet generator from the toolbar.

This text editor allows the automatic generation of text to a petrinet model.

Requirements:
  • The text input has to be in english.

  • Before a transformation can take place, the NLP algorithm have to be firstly successfully initialized.
    This may take about 1-2 minutes.

Process:
  • Enter your text in the text box and click the "Generate"-button. After that the petrinet model will be generated.

  • Alternatively, the content of a text file can be uploaded via the "Upload"-button.

  • If the text is incorrect, it can be deleted directly using the "Delete"-button.


Pattern of the text input:
In principle, any text can be entered. The individual verbs of a sentence will be translated into transitions.
In order to form operators, the keywords "or" or "and" have to be used in the sentence.
