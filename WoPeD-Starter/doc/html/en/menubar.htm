Ribbon menubar

Unless explicitly mentioned, all menu commands refer to the active editor window. The main window menubar has the following entries:

File ribbon

  •   New - open a new, empty editor (Ctrl+N)
  •   Open - load an existing Petri net from file (Ctrl+O)
  •   Recent submenu - list of recently opened files
  •   Close - close the active editor (Ctrl+W)
  •   Save - save the currently edited Petri net to a file (Ctrl+S)
  •   Save as - save the currently edited Petri net to a file with a new name (Shift+Ctrl+S)
  •   Print - print currently edited Petri net (Ctrl+P)
  •   Export - export the current net (currently supported: PNG, JPG or BMP format)(Ctrl+E)
  •   AProMoRe Import (Shift+Ctrl+I)
  •   AProMoRe Export (Shift+Ctrl+E)

Edit ribbon

  •   Undo - undo the last action(Ctrl+Z)
  •   Redo - undo the last undone action (Shift+Ctrl+Z)
  •   Cut - cut the selected items to the clipboard (Ctrl+X)
  •   Copy - copy the selected items to the clipboard(Ctrl+C)
  •   Paste - paste the clipboard contents into the currently edited Petri net(Ctrl+V)
  •   Group - group the selected items to a compound item (Ctrl+G)
  •   Ungroup - split a previously grouped, compound item into its components(Shift+Ctrl+G)
  •   Draw a place (Ctrl+1)
  •   Draw a transition(Ctrl+2)
  •   Draw an AND-split transition (Ctrl+3)
  •   Draw an XOR-split transition(Ctrl+4)
  •   Draw an AND-join transition(Shift+3)
  •   Draw an XOR-join transition (Shift+4)
  •   Draw an XOR-split-join transition (Alt+4)
  •   Draw an AND-split-join transition (Alt+3)
  •   Draw an AND-join XOR-split transition (Ctrl+Shift+3)
  •   Draw a subprocess transition(Ctrl+5)
  •   Draw a XOR-join-AND-split transition(Ctrl+Shift+3)

Analyze ribbon

  •   Start tokengame - starts the tokengame
  •   Operator coloring - effects a Coloring for the associated operators(Ctrl+Alt+C)
  •   Semantical analysis - starts the Wizard(Ctrl+Alt+A)
  •   capacity planning - opens capacity-planningdialog(Ctrl+Alt+P)
  •   Quantitative analysis - starts the quantitative analysis(Ctrl+Alt+Q)
  •   Coverability Graph - starts the coverability graph(Ctrl+Alt+R)
  •   Show metrics - opens the Process-Metrics Sidebar (Ctrl+Alt+M)
  •   Metrics mass analysis - opens the Process-Metrics mass analysis
  •   Metrics builder - opens the Process-Metrics builder
  •   Text2Process - opens the Text2Process editor

View ribbon

  •   Flip modeling direction - switches the modeling direction between vertical and horizontal(Ctrl+M)
  •   Optimize layout (multiple executions lead to different results)(Shift+Ctrl+O)
  •   Cascade windows - show all open editor windows overlapping
  •   Arrange windows - show all open editor windows without overlapping
  •   Overview - show the overview in the sidebar
  •   Treeview - show the treeview in the sidebar

Options & Help ribbon

  •   Configuration - open configuration dialog for global settings
  •   Manual - open help browser start page (Ctrl+F1)
  •   Contents - open help browser contents overview (Ctrl+Alt+F1)
  •   Sample nets - list of sample files (to be opened read-only)
  •   Report Bug - open bug report page
  •   About - open about dialog

Community ribbon

  •   Register - open registration dialog on WoPeD website
  •   Community - open forum on WoPeD website
  •   Facebook - open WoPeD Facebook website
  •   Google+ - open WoPeD Google+ website
  •   Twitter - open WoPed Twitter website
