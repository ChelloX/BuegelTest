Resource object definiton

In the left part of the resource editor tab, resource object (individual resources) can be added, modified, deleted and assigned to resource roles and groups .

Add a resource object

In order to define a new resource object, click on "", enter its name in the associated input window and click on "Ok" to save it. The resource object will then appear in the left-hand list pane under "unassigned". Note that you have to choose unique names for all resource objects of your model.

Modify a resource object

In order to change an existing resource object  click on the associated item in the left-hand list pane, then click on the edit button "" . It will appear in the input window again and can be modified there. When finished, click on "Ok" to save the changes.

Delete a resource object

In order to delete an existing resource object, click on the associated item in the left-hand list pane. When you have marked the item click on "" to remove its definition.

Assign a resource object to resource role or group

In order to define the set of resource roles and groups an object belongs to, drag and drop it to each role or group you want to assign it. In order to unassign click on the item in the role or group pane an afterwards click on "" buttons to unassign.
