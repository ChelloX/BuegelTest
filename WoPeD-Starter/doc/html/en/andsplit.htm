AND-split transitions

An AND-split transition is an operator transition introduced by the class of workflow nets. It has one input place and two or more output places. It usually is the starting point of a parallel execution. According to the occurrence rule of Petri nets, the following behaviour is defined:

  • if there is at least one token on the input place, one token is removed from this place and one token is added to every output place

Creating

To create a new AND-split transition in an open editor use one of the following commands from the edit menu:

  • Left-click on the associated symbol  and then to the position where the transition shall appear
    or
  • Right-click at the position where the transition shall appear and select " AND-split" from the context menu

After this, you can fill-out the small editing box to assign a name to the new transition or leave this to the automatic naming mechanism of WoPeD.
