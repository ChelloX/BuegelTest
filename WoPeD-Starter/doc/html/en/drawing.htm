Drawing area

The WoPeD main window provides a multi-document drawing area which can contain an arbitrary number of editor windows. Each editor is able to represent one Petri net model. Exactly one editor is regarded as the active editor, i. e. it can receive commands having effect on the contained Petri net model.

A number of commands are available over the menubar.

In WoPeD, Petri nets are stored as PNML, an XML-based file format which makes it possible to share Petri nets among many other Petri net tools supporting this format.
