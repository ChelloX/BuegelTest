Metrics configuration

Metrics Usage

The "Use metrics" checkbox enables or disables metric loading at WoPeD startup.



Metrics Display

With the "Variable decimals" and the "Algorithm-specific decimals" the number of decimal places that will be displayed in the Metrics Sidebar can be regulated.



Metrics Builder

When you switch the "Show metrics names instead of IDs" checkbox the more human readable metric names in the lists of the Metrics Builder, rather than the more technical IDs, will be shown.

The "Show advanced error messages" checkbox provides the opportunity to enable more detailed and advanced error message in the Metrics Builder. Not needed for most users.
