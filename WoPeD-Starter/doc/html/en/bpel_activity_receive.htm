BPEL Activity "receive"

The activity receive accepts message data from a service partner. Optionally begins a process by creating an instance of the process.

 

The receive activity specifies the partner link it expects to receive a message from and the operation it expects the partner’s service to invoke. For an executable process, the receive must specify an input variable or variable part for the message data received.

 

To add a receive activity to the process manually:

1. In the properties view, select the following values in order:

  1. Partner Link

  2. Operation: valid operations appear in the drop-down list after you select a partner link

2. Optionally, select an variable from the drop-down menu or create one
