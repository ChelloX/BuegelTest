AProMoRe configuration

In the AProMoRe group, all specific settings for the AProMoRe connection can be edited.

Metrics usage

Check "Enable AProMoRe" to enable AProMoRe for metrics usage.

AProMoRe server settings

In this block all server settings can be customized; you can edit:

  • "Server URL"
  • "Port"
  • "Username"
  • "Check "Use proxy" to enable a proxy server specified in the following two fields:"
  • "Proxy URL"
  • "Port"
