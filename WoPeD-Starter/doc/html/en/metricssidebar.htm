Process Metrics Sidebar

General
The sidebar displays the results from the selected metric groups. These groups can be switched via the drop-down box at the top.

Groups
For each metric in the different groups you will see the name, the calculated value and a traffic light that is set depending on the defined threshold values.

Metrics
By clicking on any of those metric entries in the sidebar, a detailed view will appear that will provide a description for the metric (if one is available) as well as the threshold values based on which the traffic light icon was displayed. Additionally, the related elements for the calculation in the net will be highlighted if rules to do so are stored in the metric configuration. This is the case for all metrics whose name is displayed in blue.

View/edit/delete metric in builder
In the detailed view of each metric, you will also find an option to either display this metric in the Metrics Builder or edit it there as well as an option to delete that metric. Which options (and hence icons) are available depends on the type of metric: If it is an internal one, distributed by the WoPeD team, you can only view it, but not edit or delete it. Custom metrics however, can be manipulated via the builder or deleted via the sidebar.

Export values
In the top right corner of the sidebar, next to the close button, you have the option to export the currently calculated and displayed values from the sidebar to either a *.TXT or a *.CSV file.

Discard current highlighting
In the lower left corner of the sidebar, you will find an icon to collapse all expanded metric detail panels as well as hiding the currently applied highlighting in the net.

Link to metrics builder
In the lower left corner of the sidebar, you will also find an icon to jump into the Metrics Builder.
