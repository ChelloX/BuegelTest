AProMoRe Export

To export a process model to a Server, be sure that the Server-Informations at AProMoRe-Configurations are correct.

To export a process model, press the exportbutton In the following there will be the AProMoRe-Export-Dialogue:

 

 

1 Search function

To update an existing process model you can search for it by entering a "Process name", "ID", "Owner", "Native Type", "Domain". By clicking on an existing process model, the process data are filled automaticly. Update de process model by clicking the updatebutton. You can also update an existing process model by filling the Process data with the correct information.

2 Current Server

The AProMoRe-Server which is in use is shown here.

3 List of process models

Here are shown the existing process models on the chosen server. The listing updates automatic when using the search function.

4 Process data

If you want to export a new process model, you can fill the process data by entering "ID", "Owner", "Process name", "Native type", "Recent version". To export a process model, click the exportbutton.
