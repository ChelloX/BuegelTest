AProMoRe Import

Before importing a model from an AProMoRe server, be sure that the AProMoRe configuration settings are correct.
For import, press the import button which will open the following dialog box:

 

 

1 Search function

The search function is integrated in the 'Process List Headbar'. Just type in sub-strings of the associated attributes "process name", "ID", "owner", "native type", and "domain". The entries int the list of process models will be narrowed accordingly.

2 Current AProMoRe Server

This shows the selected AProMoRe-Server. To change it, use AProMoRe configuration settings or select a allready created server in the dropdown menu to switch servers quickly.

3 List of process models and user foldertree

Here the existing process models and their attributes on the chosen server are shown. On the left side all the created AProMoRe user folders are shown. Select a folder to filter the process list.

5 Progressbar

The progressbar is set to indeterminate whenever Woped is busy with data acquisition. If a server is offline, the progressbar will show "Server-Down".

To update the current editor contents as the selected process model (i. e. adding a new version of an existing model) click on "Update". To import the process model, just click on "Import" or double-click on a process model.
