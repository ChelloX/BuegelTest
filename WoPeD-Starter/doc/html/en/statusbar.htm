Statusbar

The statusbar of the main window shows a "taskbar icon" for each open editor. Clicking on an icon iconfies or deiconifies the associated editor window.

 

 

Each editor window has a statusbar of its own showing the current modeling direction which is also visualized by an icon, the current zooming factor which can be adapted using the adjacent zooming slider and a visual flag signalling whether the contained net is saved or not.
