Woflan interface

Woflan was developed at the TU Eindhoven and applies several analysis algorithms to a given Petri net. The most important property which can be checked by Woflan is soundness of workflow nets.

Installing Woflan

To install Woflan, download it from the Woflan homepage, unpack it to a new folder and run the setup routine. After this, there will be a file named wofapp.exe located in the target folder.
  Note that Woflan only runs on Windows operating systems. For other platforms, the analysis engine can instead be used online using the Woflan web service facility.

Running Woflan

To run Woflan with a given Petri net model, there are two possible ways:

  • Export the current Petri net to the Woflan-specific TPN format using "File -> Export" from the menubar and choose the file type "*.tpn" in the appearing file selection dialog. The resulting TPN file can be directly loaded by starting Woflan (for Windows platforms) or be handed over to the Woflan webservice (for other platforms).
  • On Windows platforms, Woflan and WoPeD can be plugged together. In WoPeD, specify the Woflan executable path in the configuration dialog. If Woflan is present and you entered the correct path, Woflan analysis of the currently edited Petri net can be called directly from WoPeD by selecting "Tools -> Woflan analysis" from the menubar oder clicking on the button on the toolbar.

For further information on Woflan and its user interface please refer to the Woflan documentation.
