Changing element attributes

Double-clicking on a net element or selecting " Properties" from the context menu opens a dialog box allowing to change all available attributes interactively. Be sure to perform the double-click on the center area of the element where the mouse cursor is shown as a pointing finger. Depending on the type of the element, the transition property dialog or the place property dialog appears.

Apart from these property dialogs, some attributes of Petri net elements can be changed directly using the following context menu commands.

Changing a name

Right-click on the element and select "  Rename" from the context menu. A small editing box appears and lets you type in the new name. To finish editing, use the <return> key.

Changing a trigger

In order to change the trigger of a transition, right-click on the transition symbol and choose the desired trigger type (" Message", " Resource", " Time", "None") from the context menu.

Adding or removing tokens

In order to add or remove a token to or from a place, right-click on the place symbol and select " Add token" or " Remove token" from the context menu. Through this option, tokens can only be added and removed one-by-one, so in order to add or remove more than one token, the described operation must be applied repeatedly or the above property dialog must be used.
