WoPeD Handbuch

WoPeD (Workflow Petri net Designer) ist ein einfach zu bedienendes, plattformunabhängiges Softwaretool zur Modellierung und Analyse von Geschäftsprozessbeschreibungen in Form spezieller Petrinetze, so genannter Workflow-Netze.

Herunterladen von WoPeD

Die neueste Version von WoPeD ist im Web entweder auf der WoPeD-Homepage oder auf der Sourceforge-Projektseite erhältlich. Bevor WoPeD installiert wird, muss die Java Runtime Environment (JRE - Version 1.6 oder höher) auf dem Zielrechner installiert werden. Java ist auf der Sun Java Homepage verfügbar.

Installieren von WoPeD

Nun kann es losgehen. Das herunter geladene Archiv auspacken und das enthaltene ausführbare Programm WoPeD-install-windows-.exe (im Falle eines Windows-basierten Systems) bzw. WoPeD-install-linux (im Falle eines Unix-basierten Systems) bzw. WoPeD-install-mac.pkg aufrufen. In allen Fällen wird ein "Wizard" durch den Installationsvorgang führen, bitte einfach den dortigen Instruktionen folgen.

Starten WoPeD

Nach erfolgreicher Installation befinden sich Verknüpfungen zum Starten von WoPeD auf dem Desktop und im Startmenü (zumindest im Falle Windows-basierter Systeme und der meisten Unix-basierten GUI-Systeme). Bei Mac-Systemen befindet sich ein Start-Icon im Programme-Ordner. Alternativ kann WoPeD auch durch direktes Aufrufen oder Doppelklicken der Datei WoPeD.exe bzw. WoPeD (im Falle von Unix) bzw. WoPeD.app (im Falle von Mac) im Installationsordner gestartet werden. Es kann auch sinnvoll sein, WoPeD als Anwendung an die Dateierweiterung "*.pnml" zu binden, womit das Programm direkt durch Doppelklick auf die PNML-Datei mit der entsprechenden geöffneten Datei gestartet wird. Die ist bei Mac schon voreingestellt.

Verwenden von WoPeD

Nach dem Start erscheint das Hauptfenster, das die folgenden Komponenten enthält:

  • Menübar
  • Zeichenfläche
  • Statusbar
